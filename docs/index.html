What was tested

Three realistic scale points on medium hardware, plus a P90 page-size tuning comparison at extreme alert density.

Environment

32 GB Elasticsearch / 16 GB Kibana, stack 9.4.0-SNAPSHOT, pageSize=10,000 for Scenarios A, B, and C.

Methodology

Warmup first, then two measured runs per scenario. All results below use the warm Run 2 number as steady state.

Main readout. Runtime scales with pages x per-page cost. Across the realistic matrix, per-page cost stayed in the 22-33s range; the only true outlier was the 1.9M-alert P90 reference.
