# Privileged User Monitoring Commands

## `privileged-user-monitoring` (`privmon`)

Interactive generator for privileged user monitoring datasets.

### Usage

```bash
yarn start privileged-user-monitoring [--space <space>]
```

## `privmon-quick` (`privileged-user-monitoring-quick`, `quickmon`)

Run a fast privileged user monitoring generation flow with defaults.

### Usage

```bash
yarn start privmon-quick [--space <space>] [--all]
```
